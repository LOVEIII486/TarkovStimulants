## 1.定义针剂buff

首先定义针剂对应的buff，不论有无具体效果都先定义

```csharp
public static readonly QuackBuffDefinition ETGc_Buff =
new QuackBuffDefinition(new QuackBuffFactory.BuffConfig("TarkovStimulants", "eTGc_Buff", 999001, 15f,
GetIconPath("eTG-c")))
.AddCustomLogic(new RegenerationLogic(0.08f, 0.5f, -1, true))
.AddEffect(new AttributeModifierEffect(ModifierKeyConstant.Stat.EnergyCost, 2f, true))
.AddEffect(new AttributeModifierEffect(ModifierKeyConstant.Stat.WaterCost, 2f, true));
```

你有以下可选用的逻辑

AddCustomLogic 中可选

- public RegenerationLogic(float healPercent, float interval, float healDuration = -1f, bool showUI = true)
- public EnegyWaterRestoreLogic(float energyPerTick, float waterPerTick, float interval = 1.0f)
- public DelayedBuffLogic(string targetBuffName, float delay)
- public BuffImmunityLogic(bool removeOneLayer, params int[] buffIDs)

AddEffect 可选

- public AttributeModifierEffect(string attributeName, float value, bool isMultiplier = false)

具体attributeName你必须使用类似ModifierKeyConstant.Stat.EnergyCost的格式。


## 2.定义针剂物品

接下来定义针剂物品，物品命名必须使用Stim_Morphine类似格式，所有weight默认0.1f，maxStackCount默认3。
你必须定义类似形式的UsageData，其中actionSound，useSound，useTime全部保持与默认参考代码一致。
接着在behaviors添加具体针剂的效果。
以下data所有针剂都必须添加！：
new QuackAddBuffData { buffName = "TarkovStimulants_Morphine_Buff", chance = 1.0f },

其余data根据你的设计添加
可以包括
FoodData，AddBuffData，RemoveBuffData

所有针剂的shopdata都保持与默认值一致。
```csharp
        public static readonly QuackItemDefinition Stim_Morphine = new QuackItemDefinition
        {
            BaseData = new ItemData
            {
                itemId = 999008,
                order = 107,
                localizationKey = "Stim_Morphine",
                localizationDesc = "Stim_Morphine_Desc",
                weight = 0.1f,
                value = 800,
                quality = 3,
                displayQuality = DisplayQuality.Blue,
                maxStackCount = 3,
                tags = new List<string> { "Medic", "Injector" },
                spritePath = "items/Morphine.png",
                usages = new UsageData
                {
                    actionSound = "SFX/Item/use_syringe",
                    useSound = "SFX/Item/use_syringe_success",
                    useTime = 0.5f,
                    behaviors = new List<UsageBehaviorData>
                    {
                        new FoodData { energyValue = -5f, waterValue = -10f },
                        new QuackAddBuffData { buffName = "TarkovStimulants_Morphine_Buff", chance = 1.0f },
                        new RemoveBuffData { buffID = 1001 },
                        new AddBuffData { buff = 1019, chance = 1f },
                        new AddBuffData { buff = 1083, chance = 1f },
                    }
                }
            },
            Shop = new QuackItemDefinition.ShopConfig
            {
                MerchantID = MerchantIDs.Mud,
                MaxStock = 3,
                PriceFactor = 1.0f,
                Probability = 1.0f,
                ForceUnlock = true
            }
        };
```
